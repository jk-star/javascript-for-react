# JavaScript Chapter Wise Notes

## Run Project Using Live Server (Ritwik Day) in VS Code

This project contains chapter-wise HTML notes in Markdown format.

### Requirements
- VS Code
- Live Server Extension

### Steps to Run
1. Open the project folder in VS Code.
2. Install the **Live Server** extension (if not already installed).
3. Open `index.html`.
4. Right-click on the file.
5. Click **Open with Live Server**.
